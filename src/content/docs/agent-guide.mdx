# Agent Guide

The Agent page is your main interface for interacting with Chloe.

## Chat

Type messages in the chat input to talk with Chloe. She can help with:

- General questions and brainstorming
- Solana and DeFi topics
- Code-related queries
- Fun conversation

## Terminal

The built-in terminal supports these commands:

- `help` - Show available commands
- `whoami` - Display identity
- `status` - System status
- `mood` - Chloe's current mood
- `swap <amount> <from> <to>` - Simulate a swap
- `echo <text>` - Echo text
- `date` - Current date/time
- `clear` - Clear terminal

## AMM Simulator

Use the APY calculator to estimate yields:

1. Enter your deposit amount
2. Set the APR percentage
3. Choose the time period
4. View estimated returns

## Live2D Expressions

Click the expression buttons below the Live2D model to trigger different facial expressions.
