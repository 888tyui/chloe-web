# Getting Started

Welcome to CHLOE! This guide will help you get up and running quickly.

## Prerequisites

- Node.js 18+
- PostgreSQL database
- Solana wallet (Phantom or Solflare)

## Installation

1. Clone the repository
2. Install dependencies with `npm install`
3. Copy `.env.example` to `.env` and fill in your values
4. Run `npx prisma db push` to set up the database
5. Start the dev server with `npm run dev`

## Environment Variables

| Variable | Description | Required |
|----------|-------------|----------|
| DATABASE_URL | PostgreSQL connection string | Yes |
| AUTH_SECRET | NextAuth secret key | Yes |
| OPENAI_API_KEY | OpenAI API key for AI features | No (uses mock) |
| NEXT_PUBLIC_SOLANA_RPC_URL | Solana RPC endpoint | No (uses public) |

## Quick Tour

- **Agent**: Chat with Chloe, use the terminal, explore AMM pools
- **DEX**: Swap tokens on Solana via Jupiter V6
- **Coder**: Generate code with AI and see live previews
- **Labs**: Experimental features and playgrounds
